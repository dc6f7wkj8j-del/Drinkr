Drinkr Web App

How to use on iPhone:
1. Upload this folder to any static web host such as Netlify, Vercel, GitHub Pages, or your own website.
2. Open the hosted URL in Safari on iPhone.
3. Tap Share > Add to Home Screen.
4. Open Drinkr from your Home Screen.

For a quick local preview on Mac:
1. Open Terminal in this folder.
2. Run: python3 -m http.server 8080
3. Open http://localhost:8080 in your browser.

Note: iPhone Safari cannot install a web app directly from a ZIP file. It needs to be hosted at a web URL first.
